\Dash

\DashDash

Point of entry for performing chained operations on values.

This class enables standalone Dash functions like map(), filter(), etc. to be chained together. When performing multiple operations on a single input, chaining results in clearer and shorter code.

Examples

chaining Dash\Collections\filter( Dash\Collections\map( array(1, 2, 3), function($n) { return $n * 2; } ), function($n) { return $n > 2; } ); // == array(4, 6)
